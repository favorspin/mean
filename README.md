#MEAN tutorial

##Purpose
This is a tutorial project using the MEAN stack.

##Dependencies
Node.js 0.10.33+  
MongoDB 3.0.4+

##Install
- Clone to local machine.
- Install node_modules
  - `$ npm install`
- Create Environment Config File
  - `$ touch config/env/development.js`
- Launch the server
  - `$ node server`
- Navigate to http://localhost:1337
